# Projeto_Sentinela
Sentinela Digital é um “consultor de segurança de bolso” que analisa mensagens suspeitas de WhatsApp com IA generativa na AWS. O repositório traz a arquitetura serverless, código de referência em Python (Lambda), integração com Bedrock, Rekognition e Textract, além de Terraform para provisionamento da infraestrutura.
